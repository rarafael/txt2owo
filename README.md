# txt2owo

convert text input into OwO output

### Usage
```
txt2owo [...]
```
or
```
[command] | txt2owo
```

### Installation
the default installation prefix is `/usr/local`, this can be set with the PREFIX
make option
```
git clone https://github.com/rarafael/txt2owo.git
cd txt2owo
make
sudo make install
```

### Uninstall
```
cd txt2owo
sudo make uninstall
```
